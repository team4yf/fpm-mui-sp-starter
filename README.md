# Create React Single Page App

## How to use

```sh
npm -g i fpm-dev-cli
fpm-dev create app1 -t mui-sp
cd app1
```

Install it and run:

```sh
npm install
npm run start
```

## Modify the project config

- fpmc
  - [ ] at `src/App.js` line 9

- logo
- page title
- routers/menus

## Get more info
